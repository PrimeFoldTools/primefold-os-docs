## The 5 Pillars

### Pillar 1 — Verification
Tools that help humans detect false information, AI hallucinations, and manipulated content.
- **Flagship:** FactGuard Pro
- **Core capability:** Multi-vector fact checking, hallucination detection, source credibility

### Pillar 2 — Simulation
Tools that model future outcomes, competitive responses, and decision consequences before acting.
- **Flagship:** Simulation Engine
- **Core capability:** Predictive scenario modeling, consequence mapping, risk visualization

### Pillar 3 — Cognitive
Tools that augment human thinking, reduce cognitive bias, and improve decision architecture.
- **Flagship:** Trading Blindspot Assessment
- **Core capability:** Bias detection, failure mode mapping, constraint design

### Pillar 4 — Operational
Tools that run systems, automate decisions, and maintain operating standards.
- **Flagship:** PrimeFold OS / Trading Protocol
- **Core capability:** Rules enforcement, session management, ritual automation

### Pillar 5 — Applied Intelligence (5 Domains)

| Sub-pillar | Domain | Tools |
|------------|--------|-------|
| 5A | Wealth / Trading | PrimeFold Trading, Trading Drill, Blindspot Assessment |
| 5B | Property / RENU | Deal Analyzer Pro, Tenant Screener, Creative Financing Toolkit |
| 5C | Family | StoryScape, NOM NOM, CloserConnection, WiseParent, DreamDay |
| 5D | Self / Personal | B4 Health, InSight Personalization |
| 5E | Lifestyle | SkySnipe, YieldWarp |

## Memory Law

> **Build for the person who uses it 1000 times, not the person who sees it once.**

Implications:
- Every system prompt must survive extended use, not just first impressions
- Family tools must work when operator is tired and distracted
- Trading tools must work under stress and emotional pressure
- Design for the hard use case, not the demo
